Bem-vindo ao meu blog! Este website demonstra a minha capacidade de criar websites modernos e responsivos usando técnicas de Code Vibing.

## O que é Code Vibing?

Code Vibing é sobre criar websites bonitos e funcionais com código limpo e tecnologias modernas. Este site foi construído com:

- **Next.js** - Framework React para produção
- **TypeScript** - JavaScript com tipos
- **Tailwind CSS** - Framework CSS utility-first
- **Markdown** - Para gestão de conteúdo do blog

## Funcionalidades

Este website inclui:

- Suporte multi-idioma (Inglês e Português)
- Blog com funcionalidade de pesquisa
- Integração com redes sociais
- Sistema de comentários
- Design responsivo
- Otimização SEO
